# PulseBrief Mobile

Flutter user app for PulseBrief.

## Requirements

Flutter SDK is installed at:

```text
D:\Dev\flutter
```

Enable Flutter for the current PowerShell session:

```powershell
..\scripts\use-flutter.ps1
```

## Test

```powershell
..\scripts\use-flutter.ps1
flutter test
```

## Run

```powershell
..\scripts\use-flutter.ps1
flutter run
```

The current app is the generated Flutter starter. Product screens will be implemented in the next phase.
