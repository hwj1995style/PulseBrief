## 当前可演示链路

1. Flutter 用户端可使用 mock 模式独立预览，也可切换到 Spring Boot API 模式。
2. 后端提供用户端 V1 API、Admin 候选审核 API、每日简报管理 API、采集监控 API、发布操作日志 API 和异常数据检测 API。
3. React Admin 可使用 mock 数据运行，也可配置 `VITE_ADMIN_API_BASE_URL=http://localhost:8080` 接入真实后端。
4. MySQL 可通过 `deploy/docker-compose.yml` 在宿主机 `3307` 端口启动。
